# Mockup recommendations

> The purpose of this guide is to standardise the creation of the mock-up images displayed on the website.

The file `321vegan-www/assets/mockups/iphone-air-mockup-321vegan.psd` allows you to quickly create mock-up images for display on the website.

This file can be edited using software that supports the `.psd` format; there are free options available online, such as [Photopea](https://www.photopea.com/).

Open the file `321vegan-www/assets/mockups/iphone-air-mockup-321vegan.psd` in a suitable programme, then double-click on the layer labelled `← Double Click | Your Design`.

A new window will open in which you can edit the content of the layer’s dynamic object.

Simply drag and drop your file into the red box, then resize it as you wish.

Save to apply the changes, then return to the main window.

Check that the design is correct, then export the file as a `.webp` with the following dimensions: `1080 × 1920`. Et voilà!
